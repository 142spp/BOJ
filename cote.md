## 2) lower_bound / upper_bound (범위/개수)

정렬된 배열에서:
- `lower_bound(x)`: x 이상이 처음 나오는 위치
- `upper_bound(x)`: x 초과가 처음 나오는 위치  
→ 개수 = `upper - lower`

## 자주 나오는 패턴 요약 (암기용)

- **고정 길이 K**:  
  - `sum += a[r]`, `sum -= a[r-K]`
- **가변 길이(최소 길이)**:  
  - `while (sum >= S) { ans=min(ans, r-l+1); sum-=a[l++]; }`
- **등장횟수 제한**:  
  - `cnt[a[r]]++` 후 조건 깨지면 `cnt[a[l]]--`하며 `l` 이동
- **정렬 + 투포인터**:  
  - 합/차 비교해서 `l++` 또는 `r--`
- **모노토닉 덱**:  
  - 덱 뒤에서 나보다 큰(또는 작은) 값 제거 → 덱 앞이 최솟값/최댓값
